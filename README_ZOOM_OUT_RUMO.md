# Ajuste global de densidade / Zoom out — padrão Rumo

Este pacote aplica um ajuste visual global no sistema de Concreto para reduzir a sensação de zoom excessivo e melhorar o encaixe dos elementos na tela.

## O que foi alterado

- Redução geral de escala: fontes, paddings, espaçamentos e alturas.
- Topo, sidebar e botões mais compactos.
- Filtros reorganizados para ocupar menos espaço.
- KPIs/cards com proporções menores e números protegidos contra corte.
- Gráficos com altura mais equilibrada.
- Tabelas, badges, modais e formulários compactados.
- Ajustes responsivos para desktop, tablet e celular.
- Cache bust no CSS e scripts principais para GitHub Pages carregar a versão nova.

## Segurança

Não houve alteração nas regras de negócio, Supabase, dados, autenticação ou permissões. As mudanças são visuais.

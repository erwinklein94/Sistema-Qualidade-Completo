Resumo por semana Filtro informativo usando a semana operacional quinta–quarta
Entrada de dados

Para evitar conflito de informações, esta versão não carrega dados de exemplo e não importa planilhas. Os registros devem ser criados diretamente nas telas do sistema, usando os botões de novo lançamento.

Importações bloqueadas

A importação de Excel e o carregamento de dados de demonstração foram removidos para preservar o banco como fonte única da verdade.

Se existir dado antigo salvo no navegador por versões anteriores, ele pode afetar apenas telas que ainda não foram migradas para o Supabase. Use o botão abaixo somente para limpar esse armazenamento local legado.

Como funciona agora

Banco oficial: Produção, Reprovados, Ensaios de Liberação, Painel de Séries, Dashboard e Indicador Semanal passam a usar o Supabase como fonte principal.

Entrada de dados: novos registros devem ser criados manualmente pelas telas de lançamento. A importação inicial já foi concluída e removida desta versão.

Regra operacional: o histórico oficial deve nascer dos lançamentos feitos no site, não de planilhas importadas.

# Refresh visual Rumo

Esta entrega aplica os padrões visuais da marca Rumo ao sistema de Controle de Dormentes:

- logos oficiais positivo/negativo em `assets/brand`;
- paleta institucional oficial: Azul `#003865`, Azul Claro `#32A6E6`, Verde `#1E9F7F`, Verde Claro `#7FE06C`, Amarelo `#FBD300`;
- linguagem visual de movimento com grafismos/linhas, cards arredondados, sombras e estados hover;
- login, sidebar, topo, filtros, cards, KPIs, tabelas, badges e tema escuro revisados;
- gráficos atualizados em `js/config.js` para usar a nova paleta.

Observação: a fonte principal foi definida como `Cera Pro` com fallback para `Verdana/Geneva`, porque o pacote de fontes não foi incluído nos arquivos enviados.
